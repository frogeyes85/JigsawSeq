JigsawSeq
=========
This code is for analysis of JigsawSeq data
Under development... (v.0.1)


Requirement
===========
Illumina short read data for codon-barcoded variant library (pair-end is preferred)
Memory >20Gb

perl 5.10 or higher
bwa 0.7.9a-r786 or higher
samtools 0.1.19-44428cd or higher
